What is CliMax?


CliMax is a flexible and generalizable deep learning model for climate science that can be trained using heterogeneous datasets that span different climate variables, spatiotemporal coverage, and physical grounding. CliMax extends the Transformer architecture with novel encoding and aggregation blocks that permit efficient data fusion. We pretrain CliMax using a randomized forecasting objective, and post pretraining, we can finetune the model for any downstream spatiotemporal forecasting task relating to weather and climate.
